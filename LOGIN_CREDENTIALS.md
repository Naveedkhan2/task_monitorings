## User Accounts

**There is NO default user account.** You need to create a user account by:

1. Go to the login page
2. Click **"Sign up"** link
3. Fill in:
   - Username (e.g., `john`, `mary`, `employee1`)
   - Email (e.g., `john@example.com`)
   - Password (minimum 6 characters)
4. Click **"Sign Up"**

After signing up, you'll be automatically logged in as that user.

---

## How to Create Multiple Users

1. **Login as Admin:**
